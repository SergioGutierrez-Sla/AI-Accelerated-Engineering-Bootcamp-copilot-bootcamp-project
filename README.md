# AI-Accelerated-Engineering-Bootcamp-copilot-bootcamp-project

Monorepo scaffold with a Java/Spring Boot backend and React/TypeScript frontend.

## Structure

- `backend/` – Spring Boot (Maven, Java 17)
- `frontend/` – Vite + React + TypeScript
- `.github/` – GitHub Actions workflows
- `docs/` – Architecture and feature documentation
- `prompts/` – Agent definitions and standards

## Run backend

```bash
cd backend
mvn spring-boot:run
```

## Run frontend

```bash
cd frontend
npm install
npm run dev
```
